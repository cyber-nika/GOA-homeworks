Front-end არის ვებ-პლატფორმის ის ნაწილი, რომელსაც მომხმარებელი ხედავს და ურთიერთობს. სხვა სიტყვებით, ეს არის ყველა ელემენტი, რომელიც უზრუნველყოფს ვიზუალურ გამოცდილებას — დიზაინი, ინტერფეისი, ფუნქციონალობა, რაც მომხმარებელს უნებლიე გავლენას ახდენს. Front-end მოიცავს ვებ-გვერდის ყველა გრაფიკულ ელემენტს, როგორიცაა ღილაკები, სლაიდერები, ფორმები და ყველა ის ფაქტორი, რომელიც ვიზუალურად და ინსტრუმენტულად ქმნის ინტერიერს. Front-end-ის ასახვა და განვითარება ძირითადად გამოიყენება HTML, CSS, და JavaScript-თან. Front-end დეველოპერი ქმნის ინტერფეისებს, რომლებიც მომხმარებელს შეეძლება ადვილად გამოიყენოს. 2. Back-end: Back-end არის ის ნაწილი, რომელიც არ ჩანს მომხმარებელს, მაგრამ უზრუნველყოფს ვებ-პლატფორმის მუშაობას და მმართველობას. ეს არის სისტემური ლოგიკა, მონაცემთა ბაზები, სერვერები და აპლიკაციების პროგრამირების ინტერფეისი (API). Back-end უზრუნველყოფს ვებ-გვერდის მუშაობას ისე, რომ მომხმარებელს შეუძლია მონაცემების ნახვა, განახლება, შექმნა ან წაშლა. Back-end დეველოპერი მუშაობს მონაცემთა ბაზებთან, სერვერებთან, და აპლიკაციების ლოგიკასთან. 3. Full-stack: Full-stack არის დეველოპერის ან დეველოპმენტის მიდგომა, რომელიც მოიცავს როგორც front-end-ას, ასევე back-end-ას. Full-stack დეველოპერი შეუძლია შექმნას როგორც ვებ-გვერდის ვიზუალური ნაწილი, ისე სერვერები და მონაცემთა ბაზები, რაც სრულად დაალაგებს პროექტს ერთ ადგილას. Front-end-ის 3 ძირითადი ნაწილი: HTML (HyperText Markup Language): HTML არის ვებ-გვერდების სტრუქტურის შექმნის ძირითადი ენა. იგი იძლევა საიტის ჩარჩოს და წარმოადგენს ინფორმაციას, თუ როგორ უნდა განთავსდეს ტექსტი, სურათები, ვიდეოები და სხვა ელემენტები. გამოყენება: HTML-ით აღწერილან ვებ-გვერდის სხვადასხვა ნაწილები, მაგალითად, ტექსტები, სურათები, ჰიპერლინკები, ფორმები და სხვა ინტერფეისის ელემენტები. CSS (Cascading Style Sheets): CSS გამოიყენება HTML ელემენტების სტილიზაციისთვის. ეს ენის საშუალებით შეგიძლიათ შეცვალოთ ელემენტების ფერები, ზომები, თანმიმდევრობა და სხვა ვიზუალური მახასიათებლები. გამოყენება: CSS-ს ვიყენებთ ვებ-გვერდების გრაფიკულ სტილსა და ლეაუტს — მაგალითად, რომ ყველაფერი კარგად და მარტივად აიწყოს ეკრანზე, ეცოდინოთ თითოეული ელემენტის ვიზუალური კონტროლი. JavaScript: JavaScript არის პროგრამირების ენა, რომელიც ინტერაქტიურობას იძლევა ვებ-გვერდებზე. იგი საშუალებას აძლევს დეველოპერს შექმნას დინამიური და რეაგირებადი ელემენტები, როგორიცაა პოპ-აპები, ფორმების გადამოწმება და სლაიდერები. გამოყენება: JavaScript გამოიყენება ნებისმიერი ტიპის ინტერაქტიულობისთვის, როგორიცაა ღილაკებზე დაჭერა, დინამიური შეტყობინებების გამოჩენა, ან გვერდის გადახედვისას მონაცემების მიღება სერვერიდან, მაგალითად, AJAX ტექნოლოგიის გამოყენებით.
